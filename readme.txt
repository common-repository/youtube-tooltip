"Youtube ToolTip" Click on any of your links to get a quick preview of the youtube video.

== Description ==

"Youtube ToolTip" is a simple plugin, which changes all your links of "youtube" on a fast preview for the convenience of your readers.

Modifying all the links and using "jQuery" shows an attractive & modern blog.

Also use "qTip", qTip is a tooltip plugin for the jQuery framework. It's cross-browser, customizable and packed full of features!
qTip is licensed under the open source MIT license, which allows you to use the script for whatever you want, whenever you want, however you want, as long as you include a copy of the license! 

For more information about this plugin visit http://javiergalicia.wordpress.com/2009/07/03/youtube-tooltip-to-wordpress/

== Installation ==

1. UnZip and Upload `youtubetooltip.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==
= 1. Adds this plugin use of the "jQuery library"? =
Yes, advisable not to have another plugin that uses jQuery to avoid conflicts.
